Bootstrap5 display flex practicing

The biggest difference between Bootstrap 3 and Bootstrap 4 & 5 is that Bootstrap 5 now uses flexbox, instead of floats, to handle the layout. The Flexible Box Layout Module, makes it easier to design flexible responsive layout structure without using float or positioning.

বুটস্ট্র্যাপ 3 এবং বুটস্ট্র্যাপ 4 এবং 5 এর মধ্যে সবচেয়ে বড় পার্থক্য হল যে বুটস্ট্র্যাপ 5 এখন লেআউট পরিচালনা করতে ফ্লোটের পরিবর্তে ফ্লেক্সবক্স ব্যবহার করে। নমনীয় বক্স লেআউট মডিউল, ফ্লোট বা অবস্থান ব্যবহার না করে নমনীয় প্রতিক্রিয়াশীল লেআউট কাঠামো ডিজাইন করা সহজ করে তোলে।
